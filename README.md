# Eesha AI - AI Coding Platform

Full-stack AI coding platform powered by **Kimi K2.5** via the NVIDIA API.

## Features
- AI Chat with thinking mode
- Workspace with file explorer
- Integrated terminal
- Code execution
- Streaming responses

## Environment Secrets

Set these in your Space Settings → Repository Secrets:

| Secret | Description |
|--------|-------------|
| `NVIDIA_API_KEY` | Your NVIDIA API key (required) |
| `NVIDIA_BASE_URL` | `https://integrate.api.nvidia.com/v1` |
